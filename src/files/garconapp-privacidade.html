Só de Cenoura: Política de Privacidade

Nenhum dado de usuário é coletado e enviado para canto algum. O único envio é de dados dos pedidos de mentira da App, que usam dados estáticos da própria App.

A tela de pagamento é um exemplo e nenhum dado digitado ali é manipulado de qualquer maneira. De qualquer forma, como é uma App de exemplo, você pode digitar dados de mentira lá também.

As permissões pedidas são usadas apenas nos exemplos da App. Em particular, a Câmera é usada no recurso de leitura de códigos de barras localmente para preenchimento de formulário. Nada mais é gravado ou lido pela câmera. E o próprio dado preenchido pelo código de barras não é enviado para nenhum lugar.

Essa é uma App de exemplo do curso e livro de Cordova. Você pode auditar todo código fonte no Github.
